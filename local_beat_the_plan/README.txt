beat_the_plan
par Dimitri Watel

Requis:
- python3,
- pip3
- php
- un navigateur acceptant d'exécuter du javascript


Lancement :
- extraire tous les fichiers de l'archive (on l'appellera local_beat_the_plan pour l'exemple)
- aller dans le dossier local_beat_the_plan
- installer les bibliothèques nécessaires :
> pip3 install -r requirements.txt


Sous Linux :

Pour avoir de l'aide
> python3 local_beat_the_plan.py -h

Pour tester rapidement
> python3 local_beat_the_plan.py chemin_vers_ia1.sync chemin_vers_ia2.sync ...
> cd html
> php -S localhost:8888 &
> Ouvrir index.php avec un navigateur


Sous Windows ou MacOS:
